# Contributing to SoroProtocol API

## Setup

```bash
npm install
npm run start:dev
```

## Running Tests

```bash
npm test          # unit tests
npm run test:e2e  # end-to-end tests
```

## Workflow

1. Branch from `main`: `git checkout -b feat/your-feature`
2. Write tests for new service/controller logic
3. Run `npm run lint` before opening a PR
